# GitHub Issues Tracker

A simple web application that helps developers track GitHub issues and manage tasks efficiently.

This project allows users to create, view, update, and manage issues in a clean and user-friendly interface.  
It is designed as a practice project to improve JavaScript, DOM manipulation, and basic project structure skills.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

---

## ⚙️ Features

- Create new issues
- View all issues
- Track issue status (Open / Closed)
- Assign issues to users
- Persistent data using LocalStorage
- Simple and responsive UI
